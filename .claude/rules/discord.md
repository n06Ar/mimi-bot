# Discord 応答ルール

## メッセージ受信時

Discord からメッセージを受信して調査・作業が必要な場合：

1. **まず即座に受信確認の返信を送る**（`reply` ツールで同じ `chat_id` に送信）
   - 例: 「あおー（ちょっと待ってて）」「みゃ（今見てる）」
2. 実際の作業を行う
3. **作業完了後に別の `reply` で回答を送る**

単純な質問で即答できる場合は受信確認を省略してOK。

## Claude Code チャットのミラーリング

Claude Code のチャット欄でユーザーと会話しているとき、Discord MCP が有効な場合は回答をメインチャンネルにも送信する。

- Discord MCP が無効（切断中）のときは Claude Code チャットのみで応答する（エラーにしない）
- ミラーリングは回答のみ。作業中の中間出力は送らない
